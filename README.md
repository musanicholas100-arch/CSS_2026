# CSS_2026

# Programming Language Adoption and Expertise in South Africa

This Streamlit web application analyzes programming language adoption and expertise
across selected regions in South Africa. It allows users to:
- View researcher profile information
- Upload and explore academic publications
- Visualize publication trends
- Examine programming language usage and expertise
- Explore regional distribution across provinces

## Research Focus
**Title:** A Study of Programming Language Adoption and Expertise Across Selected Regions in South Africa

**Objectives:**
- Identify which programming languages are most widely used
- Compare total users vs experts for each language
- Analyze regional distribution across South African provinces

## App Features
- **Researcher Overview:** View researcher profile and field of study
- **Publications Module:** Upload CSV of publications, filter by keyword, visualize trends
- **Programming Language Data:** See total users and experts per language
- **Regional Distribution:** View language adoption in Gauteng, Western Cape, KwaZulu-Natal, and Eastern Cape

## Technologies Used
- Python 3
- Streamlit
- Pandas
